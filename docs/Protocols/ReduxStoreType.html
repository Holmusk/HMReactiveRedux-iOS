ReduxStoreType

public protocol ReduxStoreType

This represents a Redux store that can dispatch actions to mutate internal state and broadcast state updates to subscribers.

  • State

    The app-specific state type. For example:

    struct State {
  var counter: Int = 0
  var user: User? = nil
}

    It is recommended to make the state immutable to avoid unintended side effects.

    Declaration

    Swift

    associatedtype State
  • lastState

    Get the last state instance.

    Declaration

    Swift

    var lastState: Redux.Store.LastState<State> { get }
  • dispatch

    Dispatch an action and notify listeners.

    Declaration

    Swift

    var dispatch: Redux.Store.Dispatch { get }
  • subscribeState

    Set up state callback so that every time a new state arrives, call the callback function.

    Declaration

    Swift

    var subscribeState: Redux.Store.Subscribe<State> { get }
