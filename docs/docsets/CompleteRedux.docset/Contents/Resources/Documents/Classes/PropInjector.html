PropInjector

public class PropInjector<GlobalState> : PropInjectorType

Basic Redux injector implementation that also handles view lifecycles.

  • init(store:runner:)

    Initialize the injector with a Redux store instance. Every time an injection is requested, create a new subscription to this store’s state updates, and destroy it when the injectee is disposed of.
