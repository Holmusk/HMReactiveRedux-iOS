The following type aliases are available globally.

  • ReduxReducer

    Represent a reducer that takes an action and a state to produce another state.

    Declaration

    Swift

    public typealias ReduxReducer<State> = (State, ReduxActionType) -> State
  • SubscriberID

    Unique id for a subscriber.

    Declaration

    Swift

    public typealias SubscriberID = UniqueIDProviderType.UniqueID
  • ReduxStateCallback

    Callback for state subscriptions.

    Declaration

    Swift

    public typealias ReduxStateCallback<State> = (State) -> Void
  • ReduxStateGetter

    Typealias for the state getter function.

    Declaration

    Swift

    public typealias ReduxStateGetter<State> = () -> State
  • AwaitableReduxDispatcher

    Typealias for a dispatch function whose output can be awaited on to execute immediately.

    Declaration

    Swift

    public typealias AwaitableReduxDispatcher = (ReduxActionType) -> Awaitable<Any>
  • ReduxDispatcher

    Typealias for a dispatch function that cannot be awaited on. Essentially we only send some actions into the pipeline without knowing what happens.

    Declaration

    Swift

    public typealias ReduxDispatcher = (ReduxActionType) -> Void
  • ReduxSubscriber

    Typealias for the state subscribe function. Pass in the subscriber id and callback function.

    Declaration

    Swift

    public typealias ReduxSubscriber<State> = (SubscriberID, @escaping ReduxStateCallback<State>) -> ReduxSubscription
  • ReduxUnsubscriber

    Typealias for state unsubscribe function. Pass in the subscriber id to determine which state callback should be disposed of.

    Declaration

    Swift

    public typealias ReduxUnsubscriber = (SubscriberID) -> Void
  • Navigate

    Screen navigation function.

    Declaration

    Swift

    Transformer function that takes an effect as the input and produces another effect.

    Declaration

    public typealias SagaEffectTransformer<R1, R2> = (SagaEffect<R1>) -> SagaEffect<R2>

    Transformer function that transforms one effect into another, with both effects having the same output value type.

    Declaration

    Swift
