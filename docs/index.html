ReactiveRedux

Redux implementations for iOS clients. This library provides:

  • A Rx-enabled store.
  • Middleware support and default middleware providers (e.g. router middleware);
  • Prop injection for a Redux-compatible view/view controller;
  • A side effect model (Saga) to handle asynchronous work;

For a deeper look into how this works, check out the full documentation, the repository sample app and some articles I wrote:

  • Redux for Swift (Part 1): The Basics
  • Redux for Swift (Part 2): Automatic Subscription
  • Redux for Swift (Part 3): OutProps
